Java Streams


Java Stream Parts

Source → Intermediate Operations → Terminal Operation 
List<Integer> list = Arrays.asList(1,2,3,4);

list.stream()
    .filter(x -> x % 2 == 0)
    .map(x -> x * 2)
    .collect(Collectors.toList());
Output : [4,8]

Min & Max

list.stream().min(Integer::compareTo);
list.stream().max(Integer::compareTo);
min : -1 max : 6776

Remove Duplicates

list.stream().distinct()
[1,4,2,5,6,3]
