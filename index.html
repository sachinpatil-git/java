Java Stream Examples

Author: Sachin Patil | Senior Software Engineer

1. Min & Max

List list = Arrays.asList(100,1,-1,6776);
Integer min = list.stream().min(Integer::compareTo).get();
Integer max = list.stream().max(Integer::compareTo).get();
Output: min : -1 max : 6776

2. Remove Duplicates

List list = Arrays.asList(1,4,1,2,5,2,6,3);
list.stream().distinct().collect(Collectors.toList());
Output: [1, 4, 2, 5, 6, 3]

3. Sort Ascending

list.stream().sorted().collect(Collectors.toList());
Output: [1,2,3,5,7,8,10]

4. Sort Descending

list.stream().sorted(Comparator.reverseOrder());
Output: [8,7,6,5,4,3,2,1]

5. Uppercase Strings

list.stream().map(String::toUpperCase).collect(Collectors.toList());
Output: [ARIJEETH, RAKESH, RAVI, AMOGH]

6. Sum of Elements

int sum = list.stream().mapToInt(val -> val).sum();
Output: 36

7. Any Match

list.stream().anyMatch(str -> str.equals("ravi"));
Output: true

8. Find First

list.stream().findFirst();
Output: 1

9. Even Numbers

list.stream().filter(n -> n % 2 == 0).collect(Collectors.toList());
Output: [2,4,6,8,10]

10. Count > 10

list.stream().filter(n -> n > 10).count();
Output: 4

11. All Match

list.stream().allMatch(n -> n > 5);
Output: true

12. Second Highest & Top 3

list.stream().sorted(reverseOrder()).skip(1).findFirst();
list.stream().sorted(reverseOrder()).limit(3);
Second Highest: 50
Top 3: [100, 50, 5]
