Java Stream Theory + Examples

Author: Sachin Patil | Senior Software Engineer

Java Stream - Parts

A Java Stream consists of three main parts:
  • 1. Source: Where the stream originates (Collection, Array, etc.)
  • 2. Intermediate Operations: Transformations like filter(), map(), sorted()
  • 3. Terminal Operation: Produces result like collect(), forEach(), count()
List list = Arrays.asList(1,2,3,4);

list.stream()                 // Source
    .filter(x -> x % 2 == 0)  // Intermediate
    .map(x -> x * 2)          // Intermediate
    .collect(Collectors.toList()); // Terminal
Output: [4, 8]

1. Min & Max

Used to find smallest and largest element in a collection using stream operations.
list.stream().min(Integer::compareTo);
list.stream().max(Integer::compareTo);
min : -1 max : 6776

2. Remove Duplicates

distinct() removes duplicate elements from stream.
list.stream().distinct()
[1, 4, 2, 5, 6, 3]

3. Sorting Ascending

sorted() sorts elements in natural order.
list.stream().sorted()
[1,2,3,5,7,8,10]

4. Sorting Descending

Use Comparator.reverseOrder() for descending sort.
list.stream().sorted(Comparator.reverseOrder())
[8,7,6,5,4,3,2,1]

5. Uppercase Conversion

map() transforms each element.
list.stream().map(String::toUpperCase)
[ARIJEETH, RAKESH, RAVI, AMOGH]

6. Sum of Elements

mapToInt() converts stream to IntStream and sum() calculates total.
list.stream().mapToInt(x -> x).sum()
36

7. anyMatch()

Checks if any element matches given condition.
list.stream().anyMatch(x -> x.equals("ravi"))
true

8. findFirst()

Returns first element of stream.
list.stream().findFirst()
1

9. Even Numbers Filter

filter() selects elements based on condition.
list.stream().filter(x -> x % 2 == 0)
[2,4,6,8,10]

10. Count Elements

count() returns number of elements satisfying condition.
list.stream().filter(x -> x > 10).count()
4

11. allMatch()

Checks if all elements satisfy condition.
list.stream().allMatch(x -> x > 5)
true

12. Second Highest & Top 3

Use sorting + skip/limit for ranking problems.
sorted(reverse).skip(1).findFirst()
sorted(reverse).limit(3)
Second: 50 | Top3: [100,50,5]
